## A blog feature utilizing Custom Elements

Your goal is to build a blog feature utilizing the file structure of markdwn components, and the existing codebases markdown component usage to render blog posts

1. It should use routing, so `/blog/post-1` renders the markdown file from `post-1.md`.
2. It should use the frontmatter from each of the markdown files to generate and order the blog posts.
3. `blog` should list all of the blogposts based off their title, and date
4. Each blog should be able to be loaded individual based off of routing in the `/blog` directory.
5. Each blog should be rendered in a simple black and white fashion, giving an oldschool minimalist appeal.
6. The code should be clean, and as concise as possible.
7. Review your final solution to ensure it meets all of the criteria.
